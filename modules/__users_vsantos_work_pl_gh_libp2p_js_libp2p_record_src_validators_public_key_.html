validatePublicKeyRecord

  • validatePublicKeyRecord(key: Uint8Array, publicKey: Uint8Array): Promise<void>

  • Validator for public key records. Verifies that the passed in record value is the PublicKey that matches the passed in key. If validation fails the returned Promise will reject with the error.

    Parameters

    • key: Uint8Array

      A valid key is of the form '/pk/<keymultihash>'

    • publicKey: Uint8Array

      The public key to validate against (protobuf encoded).

    Returns Promise<void>
